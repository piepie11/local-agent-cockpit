# 留痕：Web UI 中英双语 + 一键切换（默认中文）

## 做了什么
- Web UI 增加语言切换按钮（右上角）：中文 ⇄ English
  - 默认语言：中文
  - 记忆：使用 `localStorage.uiLang` 持久化选择
- UI 文案覆盖：
  - `web/index.html`：静态文案全部挂 `data-i18n*`，由前端统一渲染
  - `web/app.js`：动态渲染（列表/占位符/History 详情/SSE 状态/提示弹窗）走 `t()` 翻译
  - `web/styles.css`：topbar 右侧布局改为 flex（按钮 + token 输入同排）

## 怎么验证
- 手工：
  - 启动 `up.cmd`，打开页面
  - 点击右上角语言按钮，确认：
    - 导航/按钮/占位符文案切换生效
    - Dashboard/History/Sessions/Settings 中动态内容不会被切换覆盖丢失
- 自动：
  - `npm run m2:api:e2e` → PASS（回归 API/服务不受影响）

## 结果
- 界面默认中文，可随时切换到英文；切换后可保持当前页面/当前数据视图不丢失。
