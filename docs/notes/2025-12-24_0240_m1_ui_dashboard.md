# 2025-12-24_0240 — M1：Web UI（Dashboard/History/Sessions/Settings）

## 做了什么
- 扩展静态 Web 界面（移动端优先、无构建）：`web/index.html`、`web/app.js`、`web/styles.css`
- Dashboard：
  - workspace 下拉选择 + 基本信息（root/plan）
  - sessions 选择 + 一键创建默认 sessions（codex + prompts + sandbox）
  - run 选择/创建 + Start/Step/Pause/Stop
  - Inject（manager/executor）+ Export（md/json/jsonl，fetch+blob 下载）
  - SSE 实时事件流（events/raw），以及 manager/executor 输出聚合展示
- History：按 workspace 展示 runs 列表、过滤、点击打开
- Sessions：列表 + 创建 session（role/sandbox/systemPromptPath）
- Settings：health/config 展示 + 添加 workspace 表单

## 如何验证
1) `npm run dev`
2) 浏览器打开 `http://127.0.0.1:8787/`
3) 在 Settings 页面查看 health（会提示 allowed roots / token source）
4) 填入 ADMIN_TOKEN 后尝试添加 workspace / 创建 sessions / 创建 run

## 结果
- UI 可完成从 workspace -> sessions -> run -> 控制/导出 的闭环操作（后续再做体验增强）
