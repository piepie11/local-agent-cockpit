# 留痕：默认对外监听（HOST=0.0.0.0）

## 做了什么
- 默认监听地址从 `127.0.0.1` 改为 `0.0.0.0`：
  - `src/config.js`：`config.host` 默认值改为 `0.0.0.0`
  - `scripts/up.js`：`--host` 默认值改为 `0.0.0.0`，并更新 `--help` 文案
- 可用性提示：
  - `src/server.js`：当 `HOST=0.0.0.0/::` 时额外打印 `local: http://127.0.0.1:<port>`，避免误把 `0.0.0.0` 当成可访问地址
- 文档更新：
  - `README.md`：说明默认对外监听；本机/手机（Tailscale）如何访问；如何切回仅本机 `HOST=127.0.0.1`

## 怎么验证
- `npm run m2:api:e2e` → PASS
- 默认 host 冒烟：`PORT=0` + `HOST` 默认（0.0.0.0）启动后，用 `http://127.0.0.1:<port>/api/health` 返回 200

## 结果
- 服务默认对外监听后，本机可用 `127.0.0.1:<port>` 访问，手机可用 `http://<内网IP或Tailscale-IP>:<port>/` 访问（注意防火墙/安全配置）。
